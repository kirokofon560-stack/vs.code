# Проектирование модуля организации сессии в ВУЗе

## Описание проекта

Данный проект разработан в рамках курсовой работы на тему:

**«Проектирование модуля (системы) организации сессии в ВУЗе»**

Система представляет собой веб-приложение для организации тестирования студентов в период сессии.

Проект включает серверную и клиентскую части, реализованные с использованием современных веб-технологий.

---

## Функциональные возможности системы

На сайте реализованы следующие функции:

1. Авторизация пользователей:
   - вход администратора;
   - вход студента.

2. Просмотр расписания и списка предметов.

3. Прохождение тестирования по предметам:
   - таймер прохождения теста;
   - варианты ответов;
   - отображение результата после завершения тестирования.

4. Ведомость результатов для администратора.

5. Анимация интерфейса:
   - анимация кнопок;
   - плавное появление страниц;
   - интерактивные элементы интерфейса.

6. Ограничение повторного прохождения теста.

## Используемые технологии

- Node.js
- Express.js

### Frontend
- HTML5
- CSS3
- JavaScript

### Хранение данных
- JSON Server
- db.json

---

## Структура проекта

### Серверная часть
- `server.js` — серверное приложение
- `db.json` — база данных проекта

### Клиентская часть
- `index.html` — страница авторизации
- `dashboard.html` — расписание и предметы
- `exam.html` — страница тестирования
- `style.css` — стили интерфейса

---

## Реализованные возможности

- REST API для взаимодействия клиента и сервера
- Обработка HTTP-запросов
- Работа с тестами и результатами
- Система ролей (администратор / студент)
- Защита от повторного прохождения теста
- Адаптивный пользовательский интерфейс
